# NetherScript Documentation
Documentation for [Mapverse-net/netherscript](https://github.com/MapVerse-net/netherscript) built using VitePress

## Requirements
+ Node 24

## Style Guide
This documentation follows the below practices:

+ Each page should be marked with a heading, and description (if applicable)
+ Each subsection should be titled with a 2nd heading, and sub-subsections should be denoted by 3rd heading, each without a subtitle.
